2. chapter6
Авиарейс. Возможности: получить или изменить различную информацию
о рейсе: номер рейса, пункт назначения, марка судна, статус судна (загрузка,
разгрузка, заправка, в пути, в ремонте, готов к вылету, требуется ремонт);
узнать среднее время рейса; узнать снаряженную массу; узнать количество
топлива; отправить в пункт назначения; отремонтировать; заправить; загрузить; разгрузить; узнать хрупкость/ценность
груза. Добавить дополнительные возможности для грузового и пассажирского рейс

3. chapter7
Написать функциональный интерфейс с методом, который принимает число и возвращает булево значение. Написать реализацию такого интерфейса
в виде лямбда-выражения, которое возвращает true, если переданное число
делится без остатка на 13.
